# c-data-structures
Low-level data structures and memory management library in C featuring generic vectors, linked lists, and Small String Optimization (SSO).

# C Data Structures

Implementación educativa de estructuras de datos en C.

## Incluye

- Vector dinámico genérico
- Lista enlazada
- String con Small String Optimization (SSO)

## Características

- Manejo manual de memoria
- APIs modulares
- Estructuras opacas
- Separación .h / .c
- Complejidad temporal documentada

## Compilación

```bash
gcc -Wall -Wextra -Wpedantic -Werror -std=c11 *.c -o programa
